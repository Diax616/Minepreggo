# **Player Textures Guide**

This guide explains how to change skins for the player model in the Minepreggo mod v1.3.1.

## Setting Up

To start, the player needs to find the folder for the skin, located at:

config/minepreggo/textures/player/predefined/

This works for both singleplayer worlds and servers. Then creates a folder, named with the username at lowercase only.

Then in the named folder, put in the texture images, and name them as:

&lt;username&gt;.p&lt;number&gt;.png

Where &lt;username&gt; is the player's username at lowercase, and the number corresponding to phase number. If it's meant for no pregnancy, only the username is enough. If it's for the overlay (dressed/naked), add \_extra after the number. If it's for naked variant, add \_naked after the \_extra. For example:

minepregger.png

minepregger.p4.png

minepregger.p5_extra.png  
minepregger.p9_extra_naked.png

_Since v1.3.1, male players can also be changed, without the pregnancy phases._
