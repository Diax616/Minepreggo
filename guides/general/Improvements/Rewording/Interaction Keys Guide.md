# **Interactive Keys Guide**

This guide explains the hotkeys in the Minepreggo mod v1.3.1.

## Pregnancy Types

### Belly Rubs

Defaults as **R** key, this enables the belly rubbing state, which reduces the belly rub meter gradually. The next click disables the effect.

This is disabled during labor or miscarriage.

### Labor Push

_This is added since v1.3.0._

Defaults as **K** key, this is the key used during the labor minigame, where the player must spam it for pushing out babies.

For more information, check Pregnancy Mechanics Guide.

## Ender Power Types

### Ender Woman Teleportation

Defaults as **U** key, this allows the player to teleport while riding an ender girl. The player needs to have the crosshair at a block less than 100 blocks away for it to be available.

### Ender Power Teleportation

Defaults as **I** key, this allows the player to teleport while pregnant with an ender dragon baby. The player needs to have the crosshair at a block less than 100 blocks away, and have at least 2 ender power for it to be available.

### Dragon Fireball

Defaults as **O** key, this allows the player to shoot fireballs while pregnant with an ender dragon baby. The player needs to have at least 4 ender power for it to be available.

## Interaction Types

### Medical Checkup

Defaults as **Left Shift** key, the player needs to hold down this key while conducting a right click at a Doctor or Scientific Illager with a medical table nearby. This opens the medical checkup menu.

For more information, check Checkup Guide.

### Lying on Bed

_This is added since v1.3.1._

Defaults as **Left Alt** key, the player needs to hold down this key while conducting a right click at a bed. This enables the player to lay on bed, like vanilla Minecraft, except the player can turn their head, and it is not considered sleeping.

## Miscellaneous

### Player Texture Settings

Default as **J** key, this opens the menu for player to switch between dressed and naked variant. This works for male players.

For more information, check Player Textures Guide.
