# Release Notes: Version [1.3.4]
**Release Date:** 2026-07-01

## [Customizable Player Model]
### Patch
- **[Vanilla Body Model]**
    A minor compatibility feature allows minepreggo to avoid overwriting the player's original body to render the female body, enabling CPM to render the user model. This only works when the player is not in the pregnancy stage and the 'render female body' option is disabled. More robust compatibility will be implemented later.
